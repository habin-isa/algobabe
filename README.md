# Algobabe

### Step sequencer using 808 trap sounds pack

### Make all your 808 dreams a reality

### To run the project

- \$ git clone https://github.com/habin-isa/algobabe.git
- \$ cd algobabe
- \$ yarn
- \$ yarn start
- \$ yarn test

### Tech

- React CLI
- Pretty-quick
- Styled components
- Jest
- Axios
- Babel/EsLint
