# Algobabe

## 16 Step sequencer to make your 808 dreams reality

### Use the app

https://algobabe.herokuapp.com/

### To run the project

- \$ git clone https://github.com/habin-isa/algobabe.git
- \$ cd algobabe
- \$ yarn
- \$ yarn start
- \$ yarn test

### Tech

- React CLI
- Pretty-quick
- Styled components
- Jest
- Axios
- Babel/EsLint
- 808 trap sounds pack
